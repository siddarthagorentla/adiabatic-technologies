Adiabatic Technologies is a cutting-edge energy solutions provider specializing in high-performance lithium-ion battery packs. This repository contains the official corporate website and e-commerce platform, built with modern web technologies for a seamless, fast, and secure user experience.

---

## 🌟 Key Features

- **🚀 High-Performance Frontend**: Built using **Next.js 15 (App Router)** and **React 19** for optimized performance and SEO.
- **🎨 Modern Aesthetics**: A premium, responsive design powered by **Tailwind CSS 4**, featuring glassmorphism, smooth animations, and a focus on visual clarity.
- **🛒 Dynamic E-commerce**: Integrated product catalog with a functional **Cart system** and **Checkout process** for battery solutions.
- **🔋 Product Diversity**: Showcasing advanced LFP and NMC battery solutions for:
  - **Solar Inverters** (12V 100Ah)
  - **E-Mobility** (E-cycles & 2-Wheelers)
  - **Agricultural Equipment** (Spray Pump batteries)
- **🧮 Interactive Tools**: Built-in calculators and resource pages for battery energy estimation and technology insights.
- **📱 Fully Responsive**: Optimized for every device, from mobile phones to high-resolution desktops.

## 🏃 Getting Started

### Prerequisites

- Node.js (Latest LTS version recommended)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/siddarthagorentla/adiabatic-technologies.git
   cd adiabatic-technologies
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 📂 Project Structure

```text
├── src/
│   ├── app/            # Next.js App Router (Pages & Routes)
│   ├── components/     # Reusable UI Components
│   │   ├── layout/     # Header, Footer, Navigation
│   │   ├── sections/   # Hero, Products, CTA, etc.
│   │   └── ui/         # Base UI elements
│   ├── context/        # Global State Management (CartContext)
│   ├── data/           # Static product and resource data
│   └── styles/         # Global styles and tailwind directives
├── public/             # Static assets (images, icons, etc.)
├── next.config.ts      # Next.js configuration
├── tailwind.config.ts  # Tailwind CSS configuration
└── tsconfig.json       # TypeScript configuration
```

## 🏗️ Production Build

To create an optimized production build, run:

```bash
npm run build
```

The output will be generated in the `.next` folder. You can start the server with:

```bash
npm run start
```
